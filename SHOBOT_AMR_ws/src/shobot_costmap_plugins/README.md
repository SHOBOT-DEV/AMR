# shobot_costmap_plugins

Custom Nav2 costmap plugins (e.g., point cloud to occupancy grid).

## Launch
- `ros2 launch shobot_costmap_plugins shobot_costmap_plugins_launch.py`

Features: custom costmap layers for Nav2; see plugin configs in launch/config.

Key params: input cloud topics, frames, plugin-specific thresholds (edit launch/config).
